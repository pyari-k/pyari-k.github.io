### Goodness of fit:
All of us who would have tried out charts in spreadsheets would have tried trend lines or tried to fit lines through a set of points. Statistics is so powerful that there is a number which can predict how well the line fits into these points, 
If the goodness of fit is 1, the line fits into these points perfectly. If the number is 0, it is as good as saying that the line does not fit into the points. 
Mathematically, goodness of fit is represented as `$R^2$`. We will see why it is called that soon.  


Let us look at some data and visuals to reinforce this concept. 

**Case 1: Perfectly fitting data - Goodness of fit - 1** 

```
| Year   |      No of Videos     |  
|--------|---------------------  |
|   1    |        10             | 
|   2    |        20             | 
|   3    |        30             | 
|   4    |        40             | 
|   5    |        50             | 
```

Let the above be the number of videos generated by a youtuber in his first, second, third, fourth and fifth years of vlogging. 

If we try to plot this data in excel, this is what we get

 ![_config.yml]({{ site.baseurl }}/images/statistics/r-is-1.png)

We can see that `the trend line passes through all the points and $R^2$ is 1`. 


**Case 2: Goodness of fit close to 1 - A curve which almost fits well into the set of points**

Let the above be the number of blog posts created by a content creator in her first five months in a new job. 

```
| Month   |      No of blogs     |  
|--------|---------------------  |
|   1    |        3              | 
|   2    |        5              | 
|   3    |        7              | 
|   4    |        8              | 
|   5    |        9              | 
```
Let us try to plot a scatter plot and try to fit in a curve here and see what would be the value of R. 

 ![_config.yml]({{ site.baseurl }}/images/statistics/r-is-near-to-1.png)


This is the plot and we can see that the value of `$R^2$` is 0.97. Visually we can see that the line fits decently well within the points though not perfectly. The number matches ```math R^2``` = 0.97 which is close to 1 quantifies this.
